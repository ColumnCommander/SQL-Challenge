# SQL-Leetcode-Challenge

## Motivation

This repository contains all my SQL solutions to Leetcode database problems. I created this project to improve and solidify my SQL programming skills. Over a span of 1 month, I attempted and solved various challenges to reinforce my understanding of SQL.

## Description

The solutions are organized into three difficulty categories: Easy, Medium, and Hard. Each category contains a list of questions, followed by their respective solutions.

You can find the original questions on Leetcode's database problem set:
[Leetcode Database Problems](https://leetcode.com/problemset/database/)

## Repository Structure

**Easy:** Contains solutions to all the "Easy" level SQL problems.  
**Medium:** Contains solutions to all the "Medium" level SQL problems.  
**Hard:** Contains solutions to all the "Hard" level SQL problems.  
**Interview:** Contains solutions to all the "Interview" questions asked.

## How to Use

Browse through the categories to find problems based on difficulty level.
Each problem's solution is provided directly after the problem statement.
Feel free to clone this repository and use the solutions to practice or learn from.
